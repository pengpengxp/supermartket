* 商品分类
  这就是整个价格表与是否打折等信息，其中type最后一位为1表示95折
  (DISCOUNT)，倒数第二位为1表示买二送一(GIFT)，均为0表示普通商品
  (COMMON)

  |------------+----------+-------+------|
  | Id         | Name     | Price | Type |
  |------------+----------+-------+------|
  | ITEM000001 | 可口可乐 |  3.00 | 0x01 |
  | ITEM000002 | 羽手球   |  1.00 | 0x10 |
  | ITEM000003 | 苹果     |  5.00 | 0x00 |
  | ITEM000004 | 梨       |  3.50 | 0x00 |
